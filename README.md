## Description:
Smorebot is a project which uses a Hugging Face Le Robot arm with AI and other Hardware to make the perfect Smores!

## Features:

- Grabbing the Marshmallow with a Hugging Face Le Robot arm and placing it on the Cracker with Chocolate
- Using CAD made a rotating platform for a cracker and marshmallow to rotate on
- Uaing heat gun, hear up the chocolate and the marshmallow, making it melt
- Display temperture with a small screen
